# TMB
Tool collection to control mopidy

# Setup

* install RPI.GPIO
```
pip install RPi.GPIO
```
* install spidev
```
pip install spidev
```
* install [pi-rc522](https://github.com/ondryaso/pi-rc522)
```
pip install pi-rc522
```

* install requests
```
pip install requests
```
